# latr

A todo list for things to do latr. Native Android app and Next.js web client, synced via Firestore.

Live: https://hafaio.github.io/latr/

## Running locally

Android:

```bash
cd android
./gradlew assembleDebug
```

Or open `android/` in Android Studio.

Web:

```bash
cd web
bun install
bun dev
```
